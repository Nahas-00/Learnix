Welcome to the official Learnix documentation page. This guide explains how Learnix works, how to install it, and how to contribute or modify it.

1. Project Overview

Learnix is a web-based coding and learning platform built using PHP and MySQL. It allows users to solve categorized questions, view hints/solutions, and track their progress. Admins can manage questions, users, and system content.

2. Core Features

  • User registration & authentication
  • Admin dashboard to manage questions
  • Question bank with difficulty and topic filters
  • Hints and solutions for guided learning
  • User profile with picture and info

3. Tech Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: PHP
  • Database: MySQL

4. Installation

  1. Clone the repository or extract files into your server root (e.g., htdocs in XAMPP).
  2. Import the learnix.sql database using phpMyAdmin.
  3. Update db.php with your database credentials.
  4. Run the project via http://localhost/learnix
Tip: Ensure MySQL and Apache are running in your XAMPP/WAMP before launching the app.
